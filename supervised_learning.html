Supervised Learning

Supervised learning is a type of machine learning where a model learns to make predictions based on labeled input-output pairs. It infers a function from the provided training data that maps an input to an output. The training data includes both the input and the correct output. The model's performance is evaluated based on its ability to correctly predict the output for unseen inputs.

Equation: \( Y = f(X) \)

The equation above represents the fundamental principle of supervised learning. Here, \( X \) represents the input data, \( Y \) represents the output data, and \( f \) is the function learned by the supervised learning algorithm.
